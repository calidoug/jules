1. [Install dependencies](#installing-dependencies)
1. Launch development server: *grunt serve*

## Installing Dependencies

From within the root directory:

```sh
npm install -g yo
npm install -g generator-angular-fullstack
npm install -g bower
npm install
bower install
```
## Testing

From within the root directory:

```sh
grunt test
```
